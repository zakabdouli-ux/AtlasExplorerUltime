# AtlasExplorerUltime
## 📌 Description
Atlas Explorer est un site interactif permettant de découvrir le Maghreb (Maroc, Algérie, Tunisie) grâce à une carte dynamique et des informations touristiques détaillées.

## 🚀 Fonctionnalités
- Carte interactive avec Leaflet
- Frontières réelles (GeoJSON)
- Pays cliquables
- Villes cliquables
- Images HD
- Design moderne dark
- Itinéraire touristique

## 🛠️ Technologies
- HTML
- CSS
- JavaScript
- Leaflet

## 📁 Structure du projet
- index.html
- style.css
- script.js
- data.js
